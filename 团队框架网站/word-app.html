快速上手

本节将介绍如何在项目中使用 Nice UI

使用之前

​Nice UI 1.0是基于jquery和vue开发。在开始使用 Nice UI 之前，有必要先了解HTML、CSS 和 JavaScript 的中级知识的基础知识。

我们假设你已了解关于 HTML、CSS 和 JavaScript 的中级知识。如果你刚开始学习前端开发，将框架作为你的第一步可能不是最好的主意——掌握好基础知识再来吧！之前有其它框架的使用经验会有帮助，但这不是必需的。

引入 Nice UI

你可以引入整个 Nice UI，或是根据需要仅引入部分组件。你可以引入 Nice UI的css文件，即在入口文件main.js里引入即可。

运行

如果你的开发编辑器本身就有http预览环境（比如Hbuilder），也可以直接把Nice ui目录拖到编辑器中，直接运行。
